# Naming Convention

## Префікси ресурсів

- `ACT_` - actor prefabs
- `TPL_` - template prefabs
- `ENV_` - елементи середовища
- `HBLE_` - holdable objects
- `PROP_` - scene props
- `MAT_` - materials
- `SCN_` - scenes

## Скрипти

- назви класів: `PascalCase`
- приватні поля: `_camelCase`
- публічні властивості: `PascalCase`
- назви методів: дієслово або дієслівна фраза в `PascalCase`

## Документація

- основний файл проєкту: `README.md`
- тематичні документи: `Docs/<Topic>.md`
- назви документів мають коротко описувати один аспект проєкту
